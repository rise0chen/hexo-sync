# hexo-sync
Sync article module for [Hexo](http://hexo.io/).
you can writ on web.

## Installation
``` bash
$ npm install hexo-sync --save
```

## Usage
``` bash
$ hexo sync
```

Option | Description | Default
-------|-------------|----------
`-r`, `--rename` | rename file | null
`-c`, `--cover`  | cover file  | null

## Options
``` yaml
sync:
  Username: username
  Password: 123456
```
